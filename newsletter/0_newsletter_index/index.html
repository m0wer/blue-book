    Newsletters

    If you want to follow the meaningful changes of this site, you can either:

    • Subscribe to the RSS feed
    • Browse the Newsletter section

    RSS feed⚑

    You can choose how often you want to see the site updates:

    • Daily
    • Weekly
    • Monthly
    • Yearly

    Newsletter section⚑

    We aggregate the changes by year, month, week and day. You can navigate this section to see the latest changes.

    Credits⚑

    The newsletters and the RSS feeds are automatically created from the message commits of the repository thanks to the mkdocs-newsletter plugin.
