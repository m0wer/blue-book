### [pytest](pytest.md)

* New: Capture stdout/stderr output.

    Use the `capfd` fixture.
    
    Example:
    
    ```python
    def test_foo(capfd):
        foo()  # Writes "Hello World!" to stdout
        out, err = capfd.readouterr()
        assert out == "Hello World!"
    ```
